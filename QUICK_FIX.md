# Быстрое решение

## Правильный путь к проекту:
```bash
cd /Users/eminkupaev/Desktop/step-one-app
```

## Создать .env.local для миниапа:
```bash
cd /Users/eminkupaev/Desktop/step-one-app/miniapp
cp .env.example .env.local
```

## Установить зависимости:
```bash
# Бот
cd /Users/eminkupaev/Desktop/step-one-app/bot
npm install

# Миниап
cd /Users/eminkupaev/Desktop/step-one-app/miniapp
npm install
```

## Запустить локально:
```bash
# Бот (терминал 1)
cd /Users/eminkupaev/Desktop/step-one-app/bot
npm run dev

# Миниап (терминал 2)
cd /Users/eminkupaev/Desktop/step-one-app/miniapp
npm run dev
```

**Проект находится в:** `/Users/eminkupaev/Desktop/step-one-app`
